# Pipeline de Machine Learning para Clasificación Automática de Texto

## Descripción
Proyecto de NLP donde se implementa un pipeline completo de Machine Learning para clasificación multiclase utilizando TF-IDF y Random Forest.

## Tecnologías
- Python
- Scikit-learn
- Matplotlib
- Seaborn

## Técnicas aplicadas
- Vectorización TF-IDF
- Train/Test Split
- Random Forest
- Matriz de confusión
- t-SNE para visualización
